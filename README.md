# MasterofDeath01 Apps

## How do I install these formulae?

`brew install masterofdeath01/apps/<formula>`

Or `brew tap masterofdeath01/apps` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "masterofdeath01/apps"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
